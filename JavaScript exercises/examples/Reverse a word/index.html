Reverse a word

Type a word in the input field and click the button to see the reversed word. Using For Loop method.

The result will show here.

            
const string = "JavaScript";
let reverseStr = "";
    for (let i = string.length - 1; i >= 0; i--) {
        reverseStr += string[i];
}
console.log(reverseStr); // tpircSavaJ

Type a word in the input field and click the button to see the reversed word. Using Built-in method .split("") .reverse() .join("").

The result will show here.

            
const string = "JavaScript";
let reverseStr = string.split('').reverse().join('');
console.log(reverseStr); // tpircSavaJ
