Access Elements

This is a paragraph of text.

In this example, we use document.getElementById to access the paragraph and button elements. We then add an event listener to the button, which changes the text content of the paragraph element when clicked. Explanation The paragraph element has an id attribute set to "paragraph", which allows us to access it using document.getElementById('paragraph'). Similarly, the button element has an id attribute set to "button", which allows us to access it using document.getElementById('button'). When the button is clicked, the event listener is triggered, and the text content of the paragraph element is changed to "You clicked the button!".
