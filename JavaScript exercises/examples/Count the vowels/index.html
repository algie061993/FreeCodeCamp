Count the vowels of a word

Type a word in the input field and click the button to see the number of vowels. Using For Loop method.

The result will show here.

        
const string = "JavaScript";
let count = 0;
for (let i = 0; i < string.length; i++) {
    if (
        string[i] === "a" ||
        string[i] === "e" ||
        string[i] === "i" ||
        string[i] === "o" ||
        string[i] === "u" ||
        string[i] === "A" ||
        string[i] === "E" ||
        string[i] === "I" ||
        string[i] === "O" ||
        string[i] === "U"
    ) {
        count++;
    }
}
console.log(count); // Output: 3
